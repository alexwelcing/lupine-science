# Day in the Life — Treatment

## Hook
Three AI agents wake up, check the command center, and start shipping science before the coffee is done.

> A day in the life of the Lupine Science Hermes team.

## Story Arc
1. **Standup (Image 1 / Hook)** — The coordinator reads the command center. Three glowing agent profiles light up. The day begins.
2. **Research (Image 2)** — The researcher dives into Lean formalizations. A6 bridge specs take shape. Code and theorem proofs fill the screen.
3. **Build (Image 3)** — LUPI at lupi.live: a million-atom lattice renders in the browser. The media director frames the launch film.
4. **Ship (Image 4)** — lupine.science goes live. Articles, films, the public ledger — all flowing through agent-driven production.
5. **The Team (Image 5 / CTA)** — All three agents, one mission. A clean closing card: "Evidence before claim." Link to lupine.science.

## Visual Style
Clean dark-scientific aesthetic. Deep ink (#16171d) backgrounds with indigo (#3d4db3) glow accents and ochre (#8a5e1f) warmth. Terminal and code aesthetics. Each image feels like a window into a live agent workspace. Consistent palette across all 5 for carousel cohesion.

## Tone
Confident, precise, a little cinematic. Not hype — signal. The subtext: this is how science gets done when agents do the heavy lifting.
