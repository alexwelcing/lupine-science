Design system

Result graphics.

Publication-ready data graphics for articles, proof packs, investor decks, and research notes. Every graphic is generated from structured data as an SVG, locked to the Lupine palette and typography, and ships with source attribution.

Usage

Embed a graphic in an article with a <figure>:

<figure>
  <img src="/result-graphics/gwp-comparison.svg" alt="Refrigerant 100-year GWP comparison">
  <figcaption>IPCC AR6 · mass basis, CO₂ = 1</figcaption>
</figure>

Regenerate all graphics from source data with npm run build:graphics.
